Para publicar seu jogo no GitHub e deixá-lo acessível pelo celular ou PC:

1. **Abra o GitHub Desktop:**
   - Adicione este repositório: `C:\Users\24026434\Documents\GitHub\Plague`
   - Se os arquivos não estiverem lá, copie os arquivos desta pasta atual para lá.

2. **Copie os arquivos:**
   - Use o explorador de arquivos para mover `index.html`, `style.css`, `game.js`, `gestures.js` e a pasta `assets` para `C:\Users\24026434\Documents\GitHub\Plague`.

3. **Commit & Push:**
   - Digite um resumo (ex: "Versão inicial do Garden Defense") e clique em **Commit to main**.
   - Depois clique em **Push origin**.

4. **Habilitar GitHub Pages:**
   - No GitHub (site), vá em **Settings** > **Pages**.
   - Selecione a branch `main` e a pasta `(root)`.
   - Clique em **Save**.
   - Em poucos minutos, o link do seu jogo aparecerá lá em cima (algo como `https://stephanyaliyah.github.io/Plague/`).
